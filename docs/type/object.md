## 对象的创建

1. 对象直接量
2. 工厂函数，当需要创建大量同一类的对象，使用工厂函数，方便后续批量更改。
3. 构造函数
   1. 通过工厂函数创建的对象方法完全相同，但是却未复用，使用构造函数复用方法。
   2. 构造函数写法更加简单
      1. 自动创建空对象
      2. 自动添加复用属性和方法
      3. this指向空对象
      4. 自动返回创建的对象
