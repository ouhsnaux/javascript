# 操作符

* 数学运算 `+ - * / % **`
* 比较运算符 `=== !== > < >= <=`
* 逻辑运算符 `&&, ||, !, ??` 短路机制
  * `??` 不是 `null` 和 `undefined`
* 一元运算符 `++ --`
* 赋值运算符 `=, +=, -=`
* `...` 展开符
* 位运算 `& | ~ ^`
  * `<<` 左移
  * `>>` 右移
  * `>>>` 右移补零
* 其他
  * typeof
    * `number, string, boolean, undefined, object, function`
    * `typeof null = undefined`
    * `bigint, symbol`
  * delete
  * void
  * in, 是否包含属性，包括不可枚举属性和原型链属性
  * instanceof 是否实例
